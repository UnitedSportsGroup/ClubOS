- /[campSlug]/book

  1) Parent details (name, email, phone)
  2) Add one or multiple children:
     - child first/last name, DOB, medical/allergies/epi-pen tick/notes
  3) Select attendance:
     - choose dates (multi-select)

  4) Shows live price summary:
     - calculates per-item prices
     - applies DiscountRules based on number of bookings (interpret “bookings” as total selected sessions/items, like Friendly Manager)
  5) Submit -> creates Registration(PENDING) -> creates Stripe Checkout Session -> redirects to Stripe hosted checkout

- /[campSlug]/success?session_id=
  - Verify payment with Stripe, mark Registration PAID, create Attendance placeholders, send confirmation email
- /[campSlug]/cancel
  - Show “booking cancelled” and allow restart
